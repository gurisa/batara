# Batara 😇
Javascript framework for laravel users


❓ Motivations
---
1. Provide easy to use laravel like pattern on javascript

🖖 Usage
---
1. How to use
- Install packages: `npm install`
- Setup `.env` from `.env.example`
- Run: `npm run build`
- Enjoy

2. How to develop
- Install packages: `npm install`
- Setup `.env` from `.env.example`
- Run: `npm run dev`
- Enjoy

3. How to test
- Install packages: `npm install`
- Setup `.env.test` from `.env.example`
- Run: `npm run test`
- Enjoy
